# 表格

::: tip 语法
下面是[表格]()的语法
:::

Markdown 制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行。

## 左对齐
```
| 左对齐 | 左对齐 | 左对齐 |
| :----| :---- | :---- |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

## 右对齐
```
| 右对齐 | 右对齐 | 右对齐 |
| ----:| ----: | ----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

## 居中对齐
```
| 居中对齐 | 居中对齐 | 居中对齐 |
| :----:| :----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```
